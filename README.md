# CLI Jarvis v3.0 - Savage Cyberpunk Terminal Assistant

⚡ **No APIs. Fully offline. Pure chaos.** ⚡  

CLI Jarvis is your personal cyberpunk terminal assistant built for speed, style, and control. Designed for coders, hackers, and polymaths who want their terminal to feel alive.

---

## 🔥 Features

- **Neon Pulse Bars** for CPU, Memory, and Disk usage (live updates)  
- **Network Monitoring**: Connections, ping, and stats  
- **Git Repo Analyzer**: Current branch, commit status, dirty files  
- **DuckDuckGo HTML Search**: Query from terminal without APIs  
- **Result Browser**: Open search results directly in your default browser  
- **Keyboard-first**, API-free, fully asynchronous  
- **Tab completion** for commands  
- **Cyberpunk neon styling** for maximum hacker vibes
